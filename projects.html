Portfolio

EcoBooks is a full-stack, cross platform mobile application which allows users to trade and buy books by exercising. By using the phones inbuilt accelerometer as a pedometer, the application tracks the number of steps made and converts them into 'BookCoins', which can be used to by and trade books with other users. The firebase backend is used for all user data, and the frontend is designed in React Native.
