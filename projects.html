EcoBooks

EcoBooks is a full-stack, cross platform mobile application which allows users to trade and buy books by exercising. By using the phones inbuilt accelerometer as a pedometer, the application tracks the number of steps made and converts them into 'BookCoins', which can be used to buy and trade books with other users. Trading can be organised through our hand-build messenger to negotiate prices and organise collection. Click the home button to see a video explaining and demonstrating the app, or click the source code to see the code!

Language: Javascript / Node.js
Frontend: React Native.js | Styled Components | HTML | CSS
Backend: Firebase Authentication | Cloud Firestore
Testing: Jest

Click the home button to see a video explaining and demonstrating the app, or click the source code to see the repository!
